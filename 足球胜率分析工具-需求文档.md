# 足球比赛实时胜率分析工具 — AI 提示词文档

> **用途**：将本文档直接提供给 AI（如 CodeBuddy / ChatGPT / Claude 等），让它根据你的需求快速生成一个完整的 HTML 应用。
> **建议**：在对话开头直接粘贴此文档，并补充一句"请根据这个需求文档帮我做一个足球胜率分析网页"。

## 一、项目概述

我想做一个 **单页面 HTML 应用**，用于：
- **实时展示足球比赛信息**（比分、时间、状态等）
- **计算并可视化两队胜率**（基于多维度数据）
- **支持选择不同联赛和比赛**
- **数据自动刷新，保持实时性**

**技术栈**：纯前端 HTML + CSS + JavaScript（单文件或少量文件，无需构建工具）

---

## 二、核心功能需求

### 2.1 比赛选择器
- 下拉菜单选择联赛（如英超、西甲、意甲、德甲、法甲等）
- 下拉菜单选择该联赛下的具体比赛
- 支持搜索/筛选球队

### 2.2 实时比赛信息面板
显示选中比赛的核心信息：
| 字段 | 说明 |
|:---|:---|
| 主队名称 + 队徽 | 左侧展示 |
| 客队名称 + 队徽 | 右侧展示 |
| 实时比分 | 大号字体居中 |
| 比赛状态 | 未开始 / 进行中 / 已结束 / 暂停 |
| 比赛时间 | 开球时间、已进行时长 |
| 比赛场地 | 球场名称 |

### 2.3 胜率分析模块（核心）
基于以下维度计算两队胜率，用进度条或环形图展示：

| 维度 | 权重建议 | 数据来源 |
|:---|:---|:---|
| **近期战绩** | 25% | 近5场胜负平 |
| **联赛排名** | 20% | 当前积分榜位置 |
| **历史交锋** | 15% | 近5次交手记录 |
| **主客场表现** | 15% | 主场/客场胜率 |
| **进球/失球数据** | 15% | 场均进球、场均失球 |
| **实时状态** | 10% | 当前比分、红牌、控球率等 |

**输出形式**：
- 主队胜率 % vs 客队胜率 %
- 平局概率 %
- 各维度得分条形对比图
- 综合胜率可视化（如半圆进度条）

### 2.4 数据详情面板
- 近期战绩列表（近5场）
- 历史交锋记录
- 联赛积分榜（简化版）
- 球队关键球员信息

### 2.5 自动刷新
- 比赛进行中时，每 60 秒自动刷新数据
- 提供手动刷新按钮
- 显示"最后更新时间"

## 三、数据源与 MCP / API 方案

### 方案 A：football-data.org API（推荐，免费）
| 项目 | 说明 |
|:---|:---|
| 官网 | https://www.football-data.org |
| 免费额度 | 10 请求/分钟 |
| 认证 | 需注册获取 `X-Auth-Token` |
| 覆盖 | 欧洲五大联赛 + 欧冠 + 国家队比赛 |
| 数据类型 | 比赛、球队、球员、积分榜、射手榜 |

**可用端点**：
- `GET /competitions` — 所有联赛列表
- `GET /competitions/{id}/matches` — 联赛下所有比赛
- `GET /matches` — 当日所有比赛
- `GET /teams/{id}` — 球队详情
- `GET /competitions/{id}/standings` — 积分榜
- `GET /teams/{id}/matches` — 球队近期比赛

**请求示例**：
```http
GET https://api.football-data.org/v4/competitions/PL/matches
X-Auth-Token: YOUR_API_KEY
```

### 方案 B：API-Football（数据更丰富）
| 项目 | 说明 |
|:---|:---|
| 官网 | https://www.api-football.com |
| 免费额度 | 100 请求/天 |
| 特点 | 包含实时赔率(odds)、统计、事件、阵容 |
| 覆盖 | 全球 900+ 联赛 |

### 方案 C：自建 MCP 服务器（进阶）
如果 AI 环境支持 MCP，可使用以下开源项目（GitHub 可直接访问）：

| MCP 项目 | GitHub 地址 | 数据源 | 功能特点 |
|:---|:---|:---|:---|
| `soccer-mcp-server` | https://github.com/obinopaul/soccer-mcp-server | API-Football (RapidAPI) | 联赛、球队、球员、实时比赛、赔率数据 |
| `mcp-soccer-data` | https://github.com/yeonupark/mcp-soccer-data | SoccerDataAPI | 实时比分、阵容、事件、赔率 |

**推荐 `soccer-mcp-server`（功能最全）**
- 支持 Python 直接运行或 Docker 部署
- 提供 18+ 个工具：联赛数据、球队信息、球员统计、实时比赛、赔率等
- 安装步骤：
  ```bash
  git clone https://github.com/obinopaul/soccer-mcp-server.git
  cd soccer-mcp-server
  pip install -r requirements.txt
  export RAPID_API_KEY_FOOTBALL=你的API密钥
  python soccer_server.py
  ```
- 需要先到 https://rapidapi.com/api-sports/api/api-football 注册获取 API Key

**`mcp-soccer-data`（更轻量）**
- 专注实时比分和赔率
- 使用 `uv` 管理依赖
- 安装步骤：
  ```bash
  git clone https://github.com/yeonupark/mcp-soccer-data.git
  cd mcp-soccer-data
  uv sync
  # 创建 .env 文件写入 AUTH_KEY=你的密钥
  ```
- 需要先到 https://soccerdataapi.com/ 注册获取 Auth Key

**MCP 使用方式**：
```
// 在支持 MCP 的 AI 环境中配置好上述服务器后，可以直接用自然语言获取数据
// 例如："获取曼城近期5场比赛战绩"
// AI 会自动调用 MCP 工具并返回结构化数据
```

### 方案 D：混合方案（最实用）
1. **静态数据**：用 football-data.org 获取比赛、球队、积分榜
2. **实时比分**：用免费 websocket 或轮询 API
3. **赔率数据**：用 API-Football 的 odds 端点（如需）
4. **兜底方案**：内置模拟数据，API 不可用时展示演示效果

## 五、胜率算法建议

### 简化版算法（适合快速实现）
```javascript
function calculateWinRate(homeTeam, awayTeam) {
  let homeScore = 0;
  let awayScore = 0;

  // 1. 近期战绩 (25%)
  homeScore += homeTeam.recentForm * 0.25;  // 近5场积分率
  awayScore += awayTeam.recentForm * 0.25;

  // 2. 联赛排名 (20%)
  const totalTeams = 20; // 联赛球队数
  homeScore += ((totalTeams - homeTeam.standing) / totalTeams) * 0.20;
  awayScore += ((totalTeams - awayTeam.standing) / totalTeams) * 0.20;

  // 3. 历史交锋 (15%)
  homeScore += homeTeam.h2hWinRate * 0.15;
  awayScore += awayTeam.h2hWinRate * 0.15;

  // 4. 主客场 (15%)
  homeScore += homeTeam.homeWinRate * 0.15;
  awayScore += awayTeam.awayWinRate * 0.15;

  // 5. 攻防数据 (15%)
  homeScore += (homeTeam.goalsFor / (homeTeam.goalsFor + homeTeam.goalsAgainst)) * 0.15;
  awayScore += (awayTeam.goalsFor / (awayTeam.goalsFor + awayTeam.goalsAgainst)) * 0.15;

  // 6. 实时状态 (10%) - 仅比赛中
  if (match.status === 'IN_PLAY') {
    homeScore += (match.score.home > match.score.away ? 0.10 : 0.05) * 0.10;
    awayScore += (match.score.away > match.score.home ? 0.10 : 0.05) * 0.10;
  }

  // 归一化为概率
  const total = homeScore + awayScore;
  return {
    home: Math.round((homeScore / total) * 100),
    away: Math.round((awayScore / total) * 100),
    draw: Math.round((1 - Math.abs(homeScore - awayScore) / total) * 30) // 平局估算
  };
}
```
